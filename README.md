# github-3dgs-viewer

Minimal 3DGS viewer for displaying a local PLY file with GitHub Pages.

## Files

- `index.html`: Viewer page
- `okuamano.ply`: 3DGS PLY asset

## Usage

Open `index.html` through a web server or GitHub Pages.

If you want to display a different file, change `PLY_FILE` in `index.html`.
